PURPOSE

This application would target people who have to deal with a lot of important deadlines where their busy schedule makes it difficult for them to keep track of these tasks. Some examples of target audience include medical doctors, students, developers, engineers etc. Our To-Do list application would help them manage their tasks in a much more organized manner, with the additional advantage of getting reminders.

DISCUSSION

The application main window consists of a large 'add task' button so that users know exactly what they have to do and do not get distracted. Furthermore, we included a modal form Material UI which consists of the input fields for title, date, notes, location and radio buttons for the priority of the task. Since titles are a mandatory field for every task, a pop up appears if the user tries to save the task without a title. Next, an expandable accordion was used to display the information put in the by the users. Lastly, a checkbox and d elete button were imported from the react icons so that users can check a completed task and delete tasks from the dashboard.
